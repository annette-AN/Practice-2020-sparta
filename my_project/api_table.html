기능 요청 응답
요청 URL 요청 방식 요청 데이터 응답 데이터
보드 작성
(create)		 /api/boards POST 보드순서Id('nextId': number, 보드정보('board': [])
--> 보드정보: [보드Id('Id': number), 보드이름('name': string)])		 (JSON 형식) 'result'='success'
보드 불러오기
(read)		 GET 없음 (JSON 형식) 'result'='success'
보드 수정하기
(update)		 PUT 보드순서Id('nextId': number), 보드정보('board': [])
--> 보드정보: [보드Id('Id': number), 보드이름('name': string)])		 (JSON 형식) 'result'='success'
카테고리 작성
(create)		 /api/categorys POST 카테고리순서Id('nextId': number), 카테고리정보('category': [])
--> 카테고리정보: [보드Id('boardId': number), 카테고리Id('Id': number), 카테고리이름('name': string)])		 (JSON 형식) 'result'='success'
카테고리 불러오기
(read)		 GET 없음 (JSON 형식) 'result'='success'
카테고리 수정하기
(update)		 PUT 카테고리순서Id('nextId': number), 카테고리정보('category': [])
--> 카테고리정보: [보드Id('boardId': number), 카테고리Id('Id': number), 카테고리이름('name': string)])		 (JSON 형식) 'result'='success'
메모 작성
(create)		 /api/memos POST 메모순서Id('nextId': number), 메모정보('memo': [])
--> 메모정보: [카테고리Id('boardId': number), 메모Id('Id': number), 메모이름('name': string), 메모내용('content': string)])		 (JSON 형식) 'result'='success'
메모 불러오기
(read)		 GET 없음 (JSON 형식) 'result'='success'
메모 수정하기
